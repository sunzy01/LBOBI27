Main data set: penguins.csv
For the table: created using MS Excel

This files is an attempt to an analysis using Exploratory Data Analysis with rstudio. The dataset contains 344 observations. 

Variables:
species	- Adelie, Chinstrap, Gentoo
island - Torgersen, Dream, Biscoe 
bill_length_mm - bill length in mm
bill_depth_mm - bill depth in mm
flipper_length_mm - flipper length in mm
body_mass_g - body mass in grams
sex - male, female
